# תיאור מחלקה: ViewResultActivity

## 1. מידע כללי
*   **שם המחלקה:** `ViewResultActivity`
*   **סוג:** Activity (מסך)
*   **ייעוד:** מסך זה מציג את התוצאה הסופית של חיבור שני הגלים. הוא מראה גרף גדול ואנימטיבי המבוסס על הפרמטרים שנבחרו ב-`JunctionActivity`.
*   **אינטראקציה:** מקבלת נתונים מ-`JunctionActivity` ומשתמשת ב-`GraphView` כדי לצייר את התוצאה.

## 2. משתנים
| שם | סוג | מטרה |
| :--- | :--- | :--- |
| `graphView` | `GraphView` | אזור הציור הראשי שבו מתבצעת האנימציה. |

## 3. מתודות עיקריות
*   **`onCreate`**: מחלצת את 6 הפרמטרים מה-`Intent` ומפעילה את האנימציה ב-`GraphView`.

## 4. לוגיקה כללית
היא פשוט לוקחת את ה"מתכון" (פרמטרים) שקיבלה ומעבירה אותו ל"טבח" (`GraphView`) כדי שיכין את ה"מנה" הסופית (הגרף).

## 5. הסבר במילים פשוטות
**אנלוגיה:** דמיינו שעיצבתם מכונית בהתאמה אישית בקטלוג (`JunctionActivity`). ה-`ViewResultActivity` הוא **אולם התצוגה** שבו אתם רואים את המכונית נוסעת בפועל.
